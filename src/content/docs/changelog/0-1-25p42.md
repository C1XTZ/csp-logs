# Changelog

## New features, options and improvements

*   New functions for Weather FX, for checking which post-processing options are active;
*   Weather FX: 
    *   Option to make sun move by its original AC trajectory if date is not set, enabled by default;
    *   Option for switching your headlights automatically, disabled by default;
*   Tyres FX
    *   Set custom surface type for different tracks to fix guessing;
*   Crash with Caterham Academy is fixed;
*   Crash with disabled particles is fixed.
