# Changelog

## New features, options and improvements

*   Extra FX:
    *   Subtle fog (and distant, optionally) blur;
    *   Volumetric headlights;
    *   Option to disable certain effects, or all of them, for first person view;
*   Distant emissives fully reworked, plus, now are optional;
*   New smoke:
    *   Less smoke on start;
    *   Behavior for locked wheels improved;
    *   Works in VR;
*   New system for instruments inputs;
*   Custom analog instruments reworked;
*   Custom emissives:
    *   New `BLINK` option for emissives, for blinking indicators (finishes loop before switching off);
    *   TC-in-action indicator now blinks, as it should;
*   Freer camera: click middle mouse button while holding right mouse button to take a screenshot;
*   Support for custom description text in setup.ini;
*   Integration of FidelityFX CAS by AMD (Contrast Adaptive Sharpening) for CMAA2, custom FXAA or FXAA 3.11;
*   Mesh adjustments and shader replacements:
    *   New `LAYER` option;
    *   New `INCLUDE_IN_CUBEMAP` option (more friendly wrap for `LAYER`);
*   New extended physics inputs for difference in rotation speed between left and right wheels or either front or rear axle;
*   Extended filtering with logical expressions;
*   Weather FX: a few new functions;
*   Various fixes.
