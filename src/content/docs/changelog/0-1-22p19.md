# Changelog

## New features, options and improvements

*   Performance and lights detection fixes;
*   Different textures for different tyre materials, like so:
    ```ini
    [TYRES_FX_CUSTOMTEXTURE_M_0]
    MATERIAL=TYREWALLLEFT
    TXDIFFUSE=cars\carid\M_left.dds
    TXBLUR=cars\carid\M_left_blur.dds
    
    [TYRES_FX_CUSTOMTEXTURE_M_1]
    MATERIAL=TYREWALLRIGHT
    TXDIFFUSE=cars\carid\M_right.dds
    TXBLUR=cars\carid\M_right_blur.dds
    ```
