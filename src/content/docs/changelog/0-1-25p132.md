# Changelog

## New features, options and improvements

*   Extra FX disabled by default (that was a wrong config);
*   New material libraries are fixed;
*   Material library for tracks: more usable water and digital screens;
*   Stereo car paint shader is integrated and working;
*   Specular logic for new car paint shaders changed (documentation is coming);
*   New PBR glass shader for cars windows and lights, with optional photoelasticity effect (well, an attempt to recreate it);
*   New parallax and tesselation shaders, for now, without material library (coming soon);
*   New specular ambient for those car paint shaders by Stereo;
*   Raytracing for SSLR is rewritten to reduce holes and improve overall look, as well as to try and speed things up (reduce resolution if it's a big FPS hit);
*   Some optimizations for Extra FX;
*   HBAO+ now doesn't blur accumulation screenshots;
*   Extra FX motion blur and SSAO are much smoother with accumulation screenshots;
*   Cables thing fixed, by being disabled by default;
*   No more SSAO/HBAO+ for trees;
*   Floating shadow square in the distance fixed;
*   Shadowed wheels fixed;
*   Freezes fixed;
*   Option to alter volume of various events for a car;
*   Option to use interior sounds only;
*   Option to alter cull mode for tracks meshes;
*   Flames with accumulation blur are fixed;
*   INIpp support for PP filters (with `INCLUDE` and all that);
*   INIpp parsing improved;
*   Smaller fixes.
