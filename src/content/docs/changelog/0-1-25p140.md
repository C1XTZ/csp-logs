# Changelog

## New features, options and improvements

*   Ghost car fixed;
    *   New option to make ghost car glow;
    *   New option to force ghost car to keep its original materials;
*   Unpausing with Escape button fixed (now it reacts to unpressed button);
*   Raytracing for local reflections improved;
*   Exhaust glow fixed;
*   New option for `txEmissive` support for distant glow;
*   New option allowing to alter amount of AO from depth for displacement shader;
*   Shader `ksCarPaintSimple` is now supported as well;
*   Dark distant wheels and other similar transparency-related issues fixed;
*   Fix for water normals in material library;
*   Fix for sparks: now parameters in extension config overriding ones in data;
*   Option to use solid color for texture with shader replacements sections.
