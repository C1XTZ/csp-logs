# Changelog

## New features, options and improvements

*   Extended replays for several cars fixed;
*   Time offset from weather app doesn't get stored to replay, instead, it can be changed during replay;
*   Freer camera fixed, default speed values now closer to original ones;
*   Crash with FXAA enabled fixed;
*   Dynamic objects fixed;
*   New accumulation blur for replays (still WIP);
*   Bind extra animations to extra lights (example in E30 Drift config);
*   Bind extra animations to horn or headlights;
*   Improved executables support;
*   Tyres shader now gets a bit more of ambient shadows;
*   Configs from `loaded` folder now have lower priority;
*   Car configs finding is fixed (that issue with skins configs especially) and improved (now, it'll look for all files in subfolders with fitting names);
*   High beams got their edge back;
