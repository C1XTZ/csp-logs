## New features, options and improvements

*   A lot of Weather FX fixes;

*   Animated turning signals fixed;
*   AC's bug with sun rays in triple screen mode fixed;

*   Custom clipping distances in VR mode fixed;
*   Meshes merging fixed;
*   Multithread culling with VR fixed;
*   New option for binding clutch to a button (CM update for it is coming soon);
*   Some even smaller fixes.
