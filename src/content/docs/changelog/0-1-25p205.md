# Changelog

## New features, options and improvements

*   Issues with strange glare and black artifacts are fixed;
*   VAO is used as sun shadows in the distance (no more overlit tunnels);
*   VAO doesn't affect reflections and dynamic lights for tracks anymore;
*   Close emissive objects are dimmed for reflections to get rid of that bug with lights when camera passes close to lights;
*   New smoke should work online, in triple screen mode or in VR, or with visible helmet;
*   Distant geometry mode for some shaders to get rid of clipping;
*   Clouds in Weather FX won't appear in front of geometry anymore;
*   Better compatibility with original AC (no more of that strange issue with RSS Darche car);
*   Extra animations for cars fixed (`INPUT_AS_PROGRESS` option);
*   Trip analog odometer fixed;
*   Weather FX: option for accessing spherical harmonics allowing to improve lighting in tunnels;
*   Shader replacements:
    *   Redefine LOD in/out if needed;
    *   Replacing textures fixed;
*   New VHS filter in preparation for custom post-processing extension;
*   Other fixes and stability improvements.
