# Changelog

## New features, options and improvements

*   Extra FX: few fixes for motion buffer in single and triple, TAA rewritten from scratch to improve sharpness, reduce ghosting and improve anti-aliasing, better compatibility with trees wind animation;
*   Upscalers:
    *   FSR2/3 should be working now;
    *   Advanced upscalers are working in triple screen mode (with and without lower quality for sides, although the case with the same quality for sides still needs some improvements);
    *   Different upscale quality for sides of triple is better compatible with some upscalers;
    *   `Disable for screenshots` option is read properly;
    *   OptiScaler is downloading as it should;
*   Pixel density: new `Point` upscaler for a test, new `Catmull-Rom` upscaler (looks similar to Lanczos but faster, also added as a variant to Basic upscaler);
*   Few small adjustments for new upscalers;
*   Revenant mode: a fix for replays;
