export const allVersions = versionsData.groups.flat();
export const latestPublic = allVersions.find((v) => !v.isPreview);
export const latestPreview = allVersions.find((v) => v.isPreview);

An archive of CSP changelogs for versions that were available to the Public via patreon, acstuff.club or the CSP discord.

Changelogs are currently available for **{versionsData.counts.total}** versions, of which **{versionsData.counts.previews}** are **Previews** and **{versionsData.counts.releases}** are **Public** releases.  
The latest **Public** version <a href="/csp-logs/latest/public">{latestPublic.versionName}</a> was released <span data-timeago={latestPublic.published}>some time ago</span>, the latest **Preview** version <a href="/csp-logs/latest/preview">{latestPreview.versionName}</a> was released <span data-timeago={latestPreview.published}>some time ago</span>.

**Preview** versions are exclusive to patreons of [x4fab's Patreon](https://www.patreon.com/x4fab) and typically include additional features, most notably the RainFX module.  
**Public** versions are available for free via [acstuff.club/patch](https://acstuff.club/patch/) or directly through [Content Manager](https://acstuff.club/app/).
